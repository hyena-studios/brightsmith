## Decisions Made
1. **NormalizationResult replaces raw dict** — breaking change to classify() return type, but to_dict() provides backward compatibility. Cleaner API with typed fields.
2. **Tier renumbering** — old: tier 1/2/3/"unmapped". New: tier 1=exact, 2=prefix, 3=pattern, 4=heuristic, 0=unmapped. Cleaner integer progression.
3. **Heuristic tier separated from unmapped** — old code treated heuristic and unmapped as the same tier 3. Now heuristic is tier 4 (confidence 0.3) and truly unmapped is tier 0 (confidence 0.0). Heuristic at least matched something.
4. **Golden dataset format extended** — supports both "values" (new format with description/filters/column) and "records" (old format) keys for backward compatibility.
5. **Pipeline gate zone-specific validation** — DQ rules, golden datasets, and physical models are checked programmatically, not just as instructions. This is the "prove the human was asleep at the wheel" mechanism.
